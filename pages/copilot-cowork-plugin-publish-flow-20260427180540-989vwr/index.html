Copilot Cowork Plugin Publish Flow

This interactive page shows the launch demo story: Cowork uses a plugin skill to shape the visualization, calls a remote MCP publisher, passes through safety checks, and gets back a GitHub Pages URL ready to share.

Prompt→Skill→MCP server→Safety checks→GitHub Pages
1
Copilot Cowork
Starts from a user request such as “make a visualization of this concept and publish it.”
2
Plugin skill
Guides the model toward a self-contained HTML page with strong layout, inline assets, and safe interactions.
3
Remote MCP publisher
Accepts the full HTML document and exposes create-only publishing tools back to Cowork.
4
GitHub Pages result
Publishes an immutable page path and returns a public URL the user can open immediately.

Whole flow

The demo works best when the skill makes the page visually clear and operationally simple, then lets the MCP server handle sanitization and publishing.
